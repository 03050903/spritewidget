ColorFilter.mode(Color color, TransferMode transferMode)

Creates a color filter that applies the transfer mode given as the second argument. The source color is the one given as the first argument, and the destination color is the one from the layer being composited.

The output of this filter is then composited into the background according to the Paint.transferMode, using the output of this filter as the source and the background as the destination.
