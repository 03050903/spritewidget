Combine up to twenty values' hashCodes into one value.

If you only need to handle one value's hashCode, then just refer to its hashCode getter directly.

If you need to combine an arbitrary number of values from a List or other Iterable, use hashList. The output of hashList can be used as one of the arguments to this function.

For example:

int hashCode => hashValues(foo, bar, hashList(quux), baz);
