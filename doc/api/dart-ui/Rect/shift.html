Rect shift(Offset offset)

Returns a new rectangle translated by the given offset.

Source 

Rect shift(Offset offset) {
  return new Rect.fromLTRB(left + offset.dx, top + offset.dy, right + offset.dx, bottom + offset.dy);
}
